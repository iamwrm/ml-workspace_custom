# ml-workspace_custom

## Use github action

- docker pull mltooling/ml-workspace:0.13.2
- install spyder, docker build
- upload to github docker registry

The following does make conda work.
```
conda install -y spyder=5.1.5 qt=5.9.7 pyqt=5.9.2
```
